## Description

BuidlGuidl is a curated group of Ethereum builders shipping open-source tools, curriculum, and infrastructure. Our mission is to produce capable, security-aware developers and the public-good tooling they need.

  


## Security & Safety Work

A significant portion of our stack directly strengthens Ethereum's security posture:

  * **Speedrun Ethereum.** Free, hands-on Solidity curriculum. Our [Solidity guides library](https://speedrunethereum.com/guides) already ships deep-dives on the vulnerabilities that drain real protocols: flash-loan exploits, front-running & MEV mitigation, secure randomness, commit-reveal, ERC20 approve pitfalls, ERC4626 vault risks, NFT security best practices, and secure contract-to-contract interactions. 5,500+ builders have completed at least one challenge, 20k users registered, ~10k unique monthly visitors to the website, solidity guides, etc.
  * **Scaffold-ETH 2.** Flagship dApp toolkit (2,000+ stars, 1,300+ forks) raising the security floor for every dApp built on it: encrypted local private-key vaults for deployer keys (no more plaintext .env keys), a curated set of AI Skills for secure patterns, automated testing being wired into create-eth
  * **Hacked Wallet Recovery.** Open-source tool at [hackedwalletrecovery.com](https://hackedwalletrecovery.com/) that helps users rescue assets from compromised wallets using Flashbots.
  * **BuidlGuidl CTF.** Wargame-style Solidity security challenges. Season 1 ran in Devcon (Bangkok) and Season 2 in Devconnect (Buenos Aires), both seasons available online for anyone to complete in [https://ctf.buidlguidl.com/](https://ctf.buidlguidl.com). ~150 teams participated live.
  * **ABI Ninja.** Open-source universal contract frontend (abi.ninja). Lets any user interact directly with any EVM contract via its ABI, providing a neutral fallback when an official frontend is compromised or missing. A practical user-protection tool during incident response.
  * **BuidlGuidl Client**. Run an Ethereum node with a single command. Open-source Reth + Lighthouse node distribution with RPC proxy and monitoring, distributed to underserved regions to decentralize network infrastructure.
