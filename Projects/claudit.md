## Description

# claudit - 20,000+ audit findings as prior art for your AI

  


## What

Claudit is an MCP (Model Context Protocol) server that connects AI coding agents (such as Claude Code, Codex CLI, Cursor) to Solodit's corpus of 20,000+ real-world audit findings.

  


Through a simple set of tools, AI agents can search across the entire history of disclosed smart contract vulnerabilities, filter by severity, audit firm, protocol, language or tag, and pull the full details of any specific finding on demand.

  


In short, Claudit gives AI agents access to the Ethereum security community's collective memory.

  


## Who

Built by Martin Marchev, a Senior Security Researcher at Certora and one of 200 recipients of the ETHSecurity Badge from TheDAO Security Fund. Open source, MIT-licensed, developed in the open on GitHub.

  


## Why

AI-assisted auditing is becoming a reality but the agents keep rediscovering the same bugs in slightly different shapes. They have the reasoning ability but they lack the prior art.

  


Every year, millions of dollars in smart contract exploits are lost to bug patterns that were already documented and fixed elsewhere in the ecosystem. The knowledge exists - it just is not reaching the tools that developers and auditors are actively using.

  


claudit closes that gap. Whether you are writing a new protocol, auditing an existing one or running an AI-powered security agent, the lessons from thousands of previous audits become searchable context inside your coding environment.

  


This is a public good project. Better prior art awareness means fewer repeat bugs, safer deployments and more secure protocols on Ethereum.

claudit implements various MCP tools that the agent can call during a coding or audit session to search for prior art findings.
