## Description

We work on novel methods to make ZK technology as secure as it needs to be to host the future of privacy.

  


**Why?** Privacy is an integral part of Ethereum's future. ZK is the technology for privacy building blocks the entire ecosystem will rely on. Bugs that affect soundness and completeness of ZK systems are catastrophic when exploited. Though ZK fuzzing many of these bugs could already be found and fixed but the work is not done.

  


**Who?** Valentin Wüstholz (Principal Researcher, Consensys Diligence) together with Prof. Maria Christakis (TU Wien) and PhD researchers work tirelessly discovering bugs in fundamentally important protocols like Aztec (Noir), Gnark, Nexus, risc0, Jolt, etc.

  


**What?** Automated fuzzing of zero-knowledge circuit-processing pipelines. The output are specialised OSS fuzzers like Circuzz, published at ACM CCS 2025. The team needs financial support to continue research and create more OSS tools and educational content for ZK developers. We want to continue to build and open source these tools for the industry.

  


With $25k - $50k we would fund an extra PhD position to focus on novel theories we couldn't test yet and create educational material. 

  


With >$50k we would add an intern position on top to solely focus on running fuzzers on critical ZK infrastructure.

  


**Where?** TU Wien and Consensys Diligence, with open-source tooling and coordinated disclosure to pipeline maintainers. Code: github.com/Rigorous-Software-Engineering/circuzz
