## Description

Drew Security is a public good that provides free security education, guidelines, bots, and frameworks for Ethereum/EVM.

  


A core offering of Drew Security is our free Security Education and discord bots: The Drew Security Guard Bot and Drew Security Announcement Bot. I developed these bots and provide them at no cost to help prevent server compromises. Currently, these bots protect over 88+ servers and more than 1,500,000+ Discord community members across Web3. Without these protective measures, servers risk being compromised, and users may connect their wallets to phishing links, resulting in asset loss. You can see an example of how my bot prevented a server compromise here: <https://x.com/nft_dreww/status/1970098066272206917>

  


Drew Security also publishes free content to keep users, businesses, and the web3 community informed about the latest scam tactics to protect their wallets. This work is vital for safely onboarding new users to the ecosystem. My goal is to continue the free security education on various platforms to reach new audiences and welcome them to web3 securely. This goes the same for businesses in Web3 and having proactive security as this grows. This includes but is not limited too Security Discord Audits, X audits, Email Audits, Security Training, Security consulting, TGE Consulting, and much more!
