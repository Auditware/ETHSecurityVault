## Description

Walletbeat is an open-source public good dedicated to bringing transparency to the Ethereum wallet ecosystem. As L2BEAT has done for Layer 2s, Walletbeat does for wallets: providing objective, verifiable evaluations that help users make informed decisions.

  


The Ethereum wallet landscape lacks a neutral source of truth. Wallets compete on flashy features, not on Ethereum values, such as security, privacy, self-sovereignty, and transparency. Users have no easy way to know: does this wallet leak my personal information? Is it actually open source? Are my keys truly under my control? Walletbeat exists to answer these questions.

  


Walletbeat evaluates software wallets and hardware wallets across dozens of documented attributes following Ethereum values: security, privacy, self-sovereignty, transparency, and ecosystem. Every evaluation is backed by publicly reviewable data in our GitHub repository, cross-referenced against source code analysis (if publicly available) and data collected from wallet teams.

  


Walletbeat is committed to credible neutrality. We refuse funding from wallet-related entities. All contributor affiliations are disclosed. Our long-term governance goal is to become an independent DAO or foundation.

  


Walletbeat has received prior funding from the Ethereum Foundation, including a grant through the Pectra Proactive Grant Round and a larger grant as part of the EF's Trillion Dollar Security initiative.

  


Your funding supports continued evaluation work: filling data gaps across existing listings, onboarding new wallets, expanding hardware and embedded wallet coverage, and improving the evaluation framework. Every contribution helps keep Walletbeat comprehensive, current, and useful for Ethereum users.

  


Thank you for your support and for supporting public goods in Ethereum.
