## Description

# OSs-security: Open-source device hardening for Linux, macOS & Windows

## Who

Opsek is a Web3 operational security firm. We audit smart contracts, run OpSec training for crypto founders, investigate compromises, and protect treasuries for DeFi protocols and high-net-worth individuals. We've responded to enough incidents to know one thing: most Web3 hacks don't start on-chain. They start on someone's laptop.

  


## What

[OSs-security](https://github.com/Opsek/OSs-security) is a free, open-source toolkit of **hardening scripts and factory reset guides** for Linux, macOS, and Windows. Run a single script and your machine moves from a default consumer config to a security-conscious baseline: firewall tightened, telemetry reduced, attack surface cut, logging enabled, unnecessary services disabled. If you suspect compromise, the factory reset guides walk you through safe recovery.

The scripts have been **audited by**[**Sigma Prime**](https://github.com/Opsek/OSs-security/blob/main/audit-report-hardening-scripts.pdf) and are currently being audited by Coinspect — report lives in the repo.

  


## Where

  * Repo: <https://github.com/Opsek/OSs-security>
  * Opsek: <https://opsek.io>
  * Audit report: [PDF in repo](https://github.com/Opsek/OSs-security/blob/main/audit-report-hardening-scripts.pdf)



  


## How

Fully open-source. Bash for Linux/macOS, PowerShell for Windows. Modular and auditable — every control is readable. PRs welcome. No telemetry, nothing phoning home. Built for solo devs, multisig signers, validator operators, and anyone who handles funds from a personal machine.

  


## When — and what your donation unlocks

Live today: 100+ commits, audit complete, in active use by Opsek clients and the broader community. Matching funds will let us:

  * Expand OS coverage (more Linux distros, older macOS/Windows builds)
  * Add threat-model-specific profiles (solo dev, multisig signer, validator, high-value target)
  * Ship video walkthroughs for non-technical users
  * Fund continuous re-auditing as operating systems evolve
  * Localize guides (ES, PT, ZH, RU)



If you sign Ethereum transactions on a laptop, this is the layer most people forget. Help us keep it free, audited, and in more hands.
