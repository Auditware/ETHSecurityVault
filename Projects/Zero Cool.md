## Description

**Help make AI-powered security review more accessible to early-stage crypto projects.**

  


**Who?**

ZeroCool is an AI-powered security platform built to help find real vulnerabilities in crypto protocols. We are bootstrapped, have no outside funding, and are developing the tool with support from experienced security researchers who test it on real codebases, provide feedback, and help improve its ability to identify serious issues.

  


Zero Cool is built by security researchers (devtooligan and i-am-phaze) with a proven record in public security competitions and bug bounty work. The tool has found dozens of confirmed severity bug bounties and achieved seven top-10 finishes in public audit contests, including two 1st-place finishes and one 2nd-place finish, and is currently ranked in the top 25 on the HackenProof all-time leaderboard. 

  


**What?**

This fund will support the continued development of Zero Cool and help provide free security scans to early-stage protocols. It will also help pay security researchers to use the tool on real targets, validate results, improve detection quality, and submit valid bug bounty findings where appropriate.

  


**Where?**

The project is focused on crypto protocols and blockchain infrastructure, with an emphasis on early-stage teams that need better security insight before they have mature security programs, large treasuries, or outside funding.

  


**When?**

Funding will be used on a rolling basis to improve the Zero Cool platform, support researcher-led testing, and provide free scans to eligible early-stage protocols. As part of this initiative, ZeroCool plans to provide 100 free scanning runs to early-stage teams.

  


**Why**

Zero Cool exists to make vulnerability discovery faster, cheaper, and more scalable. By combining AI-powered scanning with human security expertise, the project helps teams identify risk areas earlier and gives researchers better tools for finding serious issues.

  


**How**

Funds donated to this project will be used to improve the Zero Cool scanning platform, including the AI agents, verification workflow, infrastructure, benchmarking, and researcher feedback loops that make the tool more effective over time.

  


Donations will also help pay security researchers to test Zero Cool on real protocols, validate findings, reduce false positives, identify missed bug classes, and contribute feedback that improves the platform. This creates a practical cycle: researchers use ZeroCool in real security work, the tool improves from that work, and the ecosystem benefits from better vulnerability discovery.
