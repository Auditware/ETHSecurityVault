## Description

EVMcrispr is an open-source toolkit for composing, simulating, and executing complex blockchain transactions safely.

  


Write human-readable scripts that batch multiple operations into single transactions, simulate them on chain forks, and verify outcomes with assertions, all before touching real funds.

  


**What you can do with EVMcrispr:**

  * **Simulate before you execute.** Fork any EVM chain (Ethereum, Arbitrum, Optimism, Base, Polygon, Gnosis, and more), run your transactions in a sandbox, manipulate state, and assert expected outcomes.
  * **Batch complex operations.** Combine token approvals, swaps, governance votes, treasury movements, and contract deployments into single atomic transactions.
  * **Interact with any smart contract.** Call any function on any contract with automatic ABI resolution, proxy detection, and type checking.
  * **Manage DAOs.** Full Aragon DAO support: install apps, manage permissions, create tokens, and execute governance proposals.
  * **Work with on-chain and off-chain data.** Read contract state, resolve ENS names, fetch HTTP APIs, parse JSON, and use the results in your scripts.
  * **Automate with logic.** Variables, conditionals (if/else), loops (for, while), user-defined functions, array operations, and string manipulation.
  * **Integrate with AI.** An MCP server and CLI enable AI agents to compose and execute blockchain transactions.



  


**Proven in production:** EVMcrispr has been adopted by a wide range of DAOs and decentralized organizations to handle governance, treasury operations, and protocol upgrades. Projects like Curve, Aragon, NFTX, 1Hive, Giveth, TECommons, BrightID, Agave, and DAppNode have relied on it to compose and execute complex on-chain votes, batching permission changes, token mints, funding allocations, contract upgrades, and even DAO chain migrations, into single, auditable transactions. For many of these communities, EVMcrispr became the standard way to safely prepare and simulate important proposals before they go to a vote, reducing the risk of costly mistakes in high-stakes DAO operations.

  


**Real-world impact:** EVMcrispr was used to simulate and encode the transaction that activated TheDAO Security Fund, a single batched call that safely moved over $220M in ETH and DAO tokens to establish Ethereum's largest security initiative.

  


EVMcrispr is free, open source (AGPL-3.0), and available at [evmcrispr.com](https://evmcrispr.com).
