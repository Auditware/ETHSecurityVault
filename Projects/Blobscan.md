## Description

# 🌐 Blobscan - Making Ethereum’s Blob Layer Transparent

  


Every day, rollups publish gigabytes of data into Ethereum blobs — the foundation of Ethereum's scaling strategy, securing billions of dollars in value.

  


But the blob layer is still largely invisible.

  


Unlike transactions or smart contract calls, blob data has no dedicated tooling for inspection or analysis. If unusual patterns emerge or something goes wrong in a rollup, there's no straightforward way for researchers or developers to investigate. A layer that can't be independently inspected can't be independently trusted.

  


Compounding this, blob data is ephemeral: Ethereum nodes prune it after 18 days by default, leaving historical data either gone entirely or locked behind centralized providers — with no guarantee any individual node retains it longer. As PeerDAS rolls out, nodes will store even less complete data — making independent archival increasingly critical.

  


**Blobscan was created to change that.**

  


# 🔍 What Blobscan Does

  


Blobscan is open-source infrastructure that permanently archives every Ethereum blob and makes it queryable by anyone.

  


Through a public explorer and API, it lets you:

  * Search blob data by versioned hash, KZG commitment, transaction hash, slot, or block number
  * Decode rollup batch contents — we integrate rollup-specific decoders so raw bytes become readable across multiple L2s
  * Analyze usage patterns with real-time dashboards covering fee markets, utilization rates, rollup leaderboards, and more
  * Access historical data programmatically via a free public API



  


Unlike general-purpose block explorers where blobs are an afterthought buried under layers of navigation, Blobscan is purpose-built for the blob layer.

  


# 🔐 Why This Matters for Security

  


Opacity is a risk. When critical infrastructure can't be independently inspected:

  * Anomalous rollup behavior goes undetected
  * Post-incident forensic analysis becomes impossible
  * The community loses the ability to verify what's actually happening on-chain



  


Blobscan makes this layer auditable. Security researchers can use it to inspect rollup batch submissions encoded in the blobs. Developers can use it to debug DA issues. Protocol teams can use it to validate client implementation and behavior after incidents.

# 🚀 What We're Building Next

  


Your donation directly helps us keep this critical infrastructure **open, decentralized, and sustainable**.

  


With your support, we will focus in the following directions:

  


## **1\. Decentralized Storage (IPFS Migration)**

  


We're moving blob archival from Google Cloud Storage to IPFS, enabling any community member to run their own IPFS node and help us replicating blob data. This removes Blobscan as a single point of failure and makes long-term preservation a shared, permissionless effort. We plan to develop a DAppNode package so node operators can contribute storage with one click.

  


## **2\. Security Tooling & Anomaly Detection**

We're building analytics and monitoring infrastructure to surface risks in real time. This includes anomaly detection for unusual blob usage patterns, APIs purpose-built for forensic investigation and post-incident analysis, and dashboards that give researchers and rollup teams continuous visibility into the health of the data availability layer.

  


## **3\. Production-Grade Analytics**

We're expanding our metrics layer with new blob-specific indicators: utilization rates, fee market trends, size distributions, rollup leaderboards, USD cost breakdowns, address-level analytics and more. All exposed via a public API and an interactive dashboard with flexible time windows and filtering.

  


## **4\. Sustained Free Access**

Infrastructure costs grow linearly with Ethereum usage. Your support keeps the core explorer and API free for researchers, developers, and the broader community — while we build towards long-term sustainability through optional paid tiers for high-volume users.

  


# 🌍  A Public Good for the Ecosystem

  


Blobscan is fully open-source (AGPL-3 / MIT). No paywalls. No gatekeeping. Core access will always be free.

  


Ethereum's future depends not just on scaling, but on keeping that scaling transparent and verifiable. That's what Blobscan is for.
