## Description

# Open-source block explorers are a vital public good

Just like we need multiple clients for Ethereum, we need multiple explorers and tooling diversity for the Ethereum ecosystem. If one goes down, the others are there to step in and support.

  


This redundancy is vital, but it is not Blockscout's mission. **Our mission is to provide a first-class, open-source, decentralized explorer for all users**. An explorer built by and for the community that can provide everything users, developers, and projects need to successfully explore and develop on a chain.

  


Every chain needs an explorer, and Blockscout is the easiest and most versatile explorer available today. Chains can easily setup the OSS version themselves, setup via a RaaS provider, or host directly with Blockscout EaaS. A growing list of chains using Blockscout are available here: <https://eaas.blockscout.com/chains-and-projects>

  


  


Over the past year we've been working hard to improve the Blockscout experience for developers and users. Thanks to community contributions, we've had the ability to optimize Blockscout for speed while overhauling the frontend for ease of use. We've introduced tools to make smart contract verification much easier, and added tagging, account functionality, marketplace features and more.

# Open-source explorers support the entire ecosystem

Ethereum Virtual Machine (EVM) blockchains and Layer 2 solutions, including zk rollups and optimistic rollups, require open-source explorers and tools to provide:

  1. Free access to data.
  2. Checks for data accuracy.
  3. Redundancy and resilience.
  4. Community-driven features, customizations, and innovations.
  5. Transparency.
  6. Quick and cost-effective implementation, particularly for testnets.
  7. Support for small and niche chains.
  8. Consistent multi-chain experience.
  9. Ongoing, rapid feature development.
  10. Trustlessness and auditability.



  


  


As we continue to decentralize the future, the tools and infrastructure supporting decentralization must also follow this pattern. We appreciate your help to support an open-source, premium block explorer - by and for the community!
