## Description

# **1\. Project Description**

CrossBar Inc, registered in DE, headquarter in Santa Clara, CA. Founded 16 years ago, CrossBar owns more than 200 patents in security hardware technology. CEO Mark Davis, PHD; CTO Sung-Hyun Jo, PHD; Chairman Ker Zhang, PHD; Chief Business Officer Elaine Wang, PHD; Chief of Hardware Security Andrew Huang, and more than 50 team members including hardware, software and cryptography experts, collaborating partners includes TSMC and Foxconn.

  


One of crossbar's team member Andrew Bunnie Huang, is a world famous open source hardware hacker, his IRIS project use infraded camera to inspect semiconductor chips, emphasizing the importance of open hardware. ([https://media.ccc.de/v/38c3-iris-non-destructive-inspection-of-silicon](https://linkprotect.cudasvc.com/url?a=https%3a%2f%2fmedia.ccc.de%2fv%2f38c3-iris-non-destructive-inspection-of-silicon&c=E,1,ISv7pUHsEAdPqnL_d0FcBGcZpPnI40b2YbL2nLNUbPmw0TGBH9plSNgNUgGigpfK5tuHdnWiWRo--BPy_g5T8k2OQYTpWxnMEHeS0qOdfvf_2DEzGPg,&typo=1)) ([https://baochip.com/](https://linkprotect.cudasvc.com/url?a=https%3a%2f%2fbaochip.com%2f&c=E,1,RHnB_-G6FW9M3CtFdGUgZSGHzMl3db1V3lxfiGuuyT2reQBvFifMKw0fVdyNWA2QwzwpmGfAPSP6W88p9sdXvVPTNULDamNStOeRXvn2x9Eq&typo=1))

  


Crossbar ([https://www.crossbar-inc.com/](https://www.crossbar-inc.com)) is building the world’s first truly open source security hardware wallet — from the silicon chip level all the way up to the application software. Unlike every existing hardware wallet on the market, which relies on proprietary, NDA-protected secure elements that users cannot inspect or verify, CrossBar’s approach opens the entire security stack for public scrutiny.

  


Our custom security processing unit combines the protection of a traditional Secure Element with the functionality of a modern microcontroller. The core cryptographic engine and data access controller are published under the CERN-OHL-W open hardware license, with source RTL available for anyone to review, test on FPGA, and verify against the physical chip. The chip features both an ARM Cortex-M7 and a RISC-V core, giving the community a verifiable execution environment from boot0 through the application layer. The chip utilizes resistive-based memory technology (ReRAM), enabling secure firmware updates without the vulnerabilities of traditional floating-gate flash, increasing durability, significantly enhancing performances such as processing power, memory size. It is the most powerful chip ever built, with 150 years of key retention, non-capacitive no scanning can extract stored secrets. Our chip developed on this ReRAM technology remains physically unbreakable, will self-destruct when sensing physical attacks in all forms.

  


This matters for Ethereum security because today’s self-custody is built on a contradiction: users choose hardware wallets to avoid trusting third parties, yet they must blindly trust opaque chip vendors whose designs are hidden behind. CrossBar eliminates this “security by obscurity” foundation by enabling genuine, verifiable self-custody.

  


Our hardware wallet ships as a compact USB-C dongle with a full feature set: OLED display, fingerprint biometric authentication, FIDO2 support, and has the optional to integrate MPC (Multi-Party Computation) capability. MPC splits cryptographic keys across multiple devices — the hardware wallet and a companion mobile app — eliminating the single point of failure that plagues traditional seed-phrase wallets.

  


The project has moved well beyond the concept stage. We have published chip-level RTL on GitHub, developed a working prototype with integrated hardware and software, and are participating in the IRIS project for unprecedented chip inspectability using low-cost laser interferometry. Our production chip incorporates Resistive Memory (ReRAM) technology, enabling secure firmware updates without the vulnerabilities of traditional floating-gate flash.

  


CrossBar directly strengthens Ethereum’s security ecosystem by providing open source security tooling and infrastructure, advancing supply chain security through chip-level transparency, implementing cryptographic primitives in verifiable hardware, and establishing new security standards and best practices for hardware wallet design and user security.

  


All development outputs are and will remain open source. We are committed to ongoing public access and will provide post-round impact reporting.

  


# 2\. Eligibility Criteria 

## 2.1 Open Source & Public Goods

### Open Source / Public Benefit

All core security components of CrossBar’s hardware wallet are open source for scrutiny. Firmware and companion software will also be released under open source licenses. Anyone can construct, inspect, or test an FPGA system that performs the same logical security operations as our production chip, and build the product that fits the most of their need.

## 2.2 Ecosystem Alignment

### Ethereum-Focused (Required)

CrossBar’s hardware wallet primarily supports Ethereum and EVM-compatible L2 security. Our software implementation targets Ethereum transaction signing, and is designed for Ethereum-native workflows. The hardware supports EIP-712 structured data signing for safe transaction verification. We are open to implementing additional Ethereum-specific features as guided by the community and the Ethereum Foundation. We focus on Ethereum clear signing for all critical applications (Safe, governance councils)

# 4\. Impact on Ethereum Security Ecosystem

## Target Security Areas Addressed

CrossBar’s work directly contributes to multiple target areas of the Ethereum Security QF Round:

  * **User Protection & Wallet Safety: **Eliminates the single point of failure in seed-phrase wallets. Open hardware enables users to verify the security of their own custody solution rather than blindly trusting vendors.
  * **Security Tooling & Infrastructure: **Provides the first open source secure element that any wallet developer can integrate, inspect, and build upon. Our FPGA test framework is reusable infrastructure for the entire hardware security community.
  * **Supply Chain Security:** Participation in the IRIS inspectability project enables physical verification that manufactured chips match the published open source design — addressing a fundamental supply chain trust problem for all hardware wallets.
  * **Cryptographic Primitives & Implementations: **Open source implementation of ECDSA and secure key storage in verifiable hardware.
  * **Security Standards & Best Practices: **Establishing new standards for transparency in hardware wallet design, moving the industry away from security-by-obscurity.



## Why This Matters Now

The Ethereum ecosystem is maturing rapidly. More value, more users, and more institutional adoption mean that the security of self-custody infrastructure is more critical than ever. Yet the hardware foundation of that security — the secure element — remains a black box across the entire industry. Certified chips have known-but-unpublished vulnerabilities hidden by NDAs. Users choosing self-custody are, paradoxically, transferring trust to an opaque platform with undisclosed flaws.

  


CrossBar addresses this structural problem at its root. By open-sourcing the silicon, we enable the kind of community scrutiny and continuous improvement that has made open source software the backbone of internet security.

  


CrossBar aims to be ready for production by November 2026.
