## Description

**_What is PRSpec?_**

  


PRSpec is an open-source tool that checks Ethereum client implementations

against EIP specifications using LLM-based semantic analysis. It doesn't

just compare text — it understands protocol intent and catches deviations

that manual review and traditional static analysis miss

  


**What has PRSpec achieved?**

  


Real-world impact:

Found a confirmed specification deviation in Nethermind's EIP-1559 implementation. A core developer acknowledged the FeeCollector property contradicts the mandatory fee burn mechanism

(<https://github.com/NethermindEth/nethermind/issues/10522)>

  


Ethereum Foundation engagement: The execution-specs team (#2 contributor) provided architectural guidance for improving PRSpec's spec extraction pipeline

(<https://github.com/ethereum/execution-specs/issues/2212)>

  


Production-ready foundation: **6 EIPs** analyzed across 3 clients (**go-ethereum, Nethermind, Besu**) in Go, C#, and Java. 62 passing tests.

Full CI pipeline. MIT licensed.

  


**Why does this need funding?**
