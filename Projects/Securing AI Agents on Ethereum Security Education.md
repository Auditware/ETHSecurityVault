## Description

**Securing AI Agents on Ethereum: Security Education & Research**

  


I'm a security researcher and educator with tutorials on decoding calldata and simulating transactions officially featured on[ ](https://ethereum.org/developers/tutorials/)[ethereum.org](http://ethereum.org)'s developer tutorials section.

This project delivers educational resources to help developers understand and mitigate risks when building and deploying AI agents on Ethereum and L2s

  


**My work:**

  


  * **BLS Lab** : Interactive toolkit for applied cryptography (BLS signatures, hashing, function selectors, and more)
  * **PQ Signature Size Explorer** : Visual comparison of post-quantum signature sizes (Dilithium, Falcon, SPHINCS+) vs classical algorithms
  * **Cyfrin tutorials** (featured on ethereum.org): Practical guides on calldata decoding and transaction simulation
  * **AI Agents & Blockspace**: Analysis of how autonomous agents compete for blockspace, MEV, and execution dynamics
  * **Transaction Simulation Best Practices for AI Agents** : In-depth guide on safe simulation and fail-closed patterns
  * **Ongoing AI × Crypto Security Research** : Focused on real-world agent risks and transaction-level vulnerabilities



  


**What the funding enables:**

  * Expand practical guides and checklists on AI agent security
  * Deeper research into emerging threats 
  * More high-quality educational content for the Ethereum community
