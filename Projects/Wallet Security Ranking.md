## Description

# What it is

The[ Wallet Security Ranking](https://www.coinspect.com/wallets/) is an objective, transparent, and reproducible evaluation of crypto wallet security across iOS, Android, and browser extensions. Built and maintained by Coinspect **since 2024** , it scores wallets across four checklists — dApp Permissions, Intent Verification, Physical Access, and Threat Prevention — using a [methodology](https://www.coinspect.com/wallets/testing/) that anyone can review and reproduce.

The ranking now covers 77 wallet builds — iOS, Android, and browser extensions tested independently, even for the same wallet. We re-test periodically as wallets ship improvements, with all changes tracked in public repositories.

# Why it matters

Users lose funds every day to phishing, wallet drainers, and malicious dApps, and they have no objective way to tell which wallet would have protected them. Marketing claims and paid listicles dominate; independent evaluation is rare. The ranking changes that. By **making security measurable and public** , it creates pressure for wallets to **ship real protections** — and gives users and the security community a shared source of truth.

# Real impact

Wallet Security Ranking is a working project used by the ecosystem, examples:

  * **MetaMask's security page** [links to the ranking](https://metamask.io/security) as a reference for users evaluating wallet security — a strong signal of industry recognition.
  * **Wallet vendors discuss with us** after shipping improvements — turning the ranking into a feedback loop that drives real security gains.
  * Other projects in the wallet-security space build on our data, integrating Wallet Security Ranking results into their own evaluations (e.g., [**Walletbeat**](https://x.com/polymutex/status/2049399101414928863)).
  * **Maintained** continuously since 2024, with three published editions to date (Oct 2024, Apr 2025, Jan 2026) — each expanding scope and refining methodology, with re-tests in between.
  * [Mainstream](https://money.com/best-crypto-wallets/#:~:text=sites%20%E2%80%94%20namely%20Certik%2C-,Coinspect,-and%20CERtified.%20These) media is starting to cite our results. Money.com's crypto wallet methodology, for example, names Coinspect as a trusted security assessment source.



# Open Resources

The Wallet Security Ranking is built on years of research, but the ranking website itself is only the visible output. The methodology, the dataset, and the checklist behind it are all open, versioned, and reusable

## Wallet Security Ranking — Public Dataset:

Every score we've ever published, with evidence, fully reproducible.

Per-wallet, per-platform check results with evidence, plus the configuration (categories, weights, included wallets) for each round. Round folders are immutable snapshots so any past ranking is fully reproducible. Licensed CC BY 4.0.

# What funding would support

**1\. Continued maintenance and expansion.** Periodic re-tests, new wallets, and new security checks as threats evolve — from phishing patterns to wallet drainers and dApp interaction risks.

**2\. More open-source tooling.** We've built several tools internally to scale testing and automate parts of it. Releasing them publicly takes additional engineering work to make them reusable by anyone.

**3\. Open the scoring weights to community votes.** Checklist weights are set by experts using an Analytic Hierarchy Process (AHP) — rigorous but tedious. A public web app would let**Ethereum Security Badge holders** and other experts vote directly, broadening who shapes the criteria.
