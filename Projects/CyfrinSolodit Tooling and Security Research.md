## Description

**Who?** Cyfrin is a leading Web3 security team building free, open-source tools that help developers write safer code and users sign transactions with confidence.

  


**What?** We build and maintain a growing suite of public goods for Web3 security:

Solodit — the #1 security research platform with 20,000+ vulnerability findings. 

Aderyn — lightning-fast Solidity static analysis. 

Moccasin — a Python-based Vyper development framework. 

Wise Signer — decoded calldata verification so you know what you're signing. 

LocalSafe — run Safe multisig wallet 100% locally. 

SolSkill — AI skills to write more secure smart contracts. 

Hardware wallet security reviews — independent, researcher-grade evaluations. 

ERC-8213 — an Ethereum standard to eliminate blind signing. 

BattleChain safe-harbor — a whitehat framework for emergency protocol rescues.

  


**Why?** Billions have been stolen through preventable hacks like Bybit ($1.4B) and Radiant ($50M). Better tooling is the fix. We keep everything free and open-source so cost is never a barrier to building and transacting securely.

  


**Where?** All online and open-source: solodit.cyfrin.io, docs.cyfrin.io, etc

  


**How?** Donations fund development, maintenance, and new tools — keeping them free forever.
