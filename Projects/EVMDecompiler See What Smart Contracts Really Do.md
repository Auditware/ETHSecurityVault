## Description

## **Who we are**

We are a team of blockchain security researchers and engineers working on **AI-powered analysis of smart contracts and blockchain systems**.

Our work has been used in both academic research and real-world security workflows, and we focus on one core mission:

👉 **making smart contracts understandable, even when the source code is not available**

##   


## **The problem**

Ethereum is transparent — but in practice, most of it is **not understandable**.

👉 **Over 99% of smart contracts are effectively closed-source** (not verified or not easily readable).

Users interact with them. Money flows through them.

But most people cannot see or understand what these contracts actually do.

This creates a dangerous gap:

  * Users trust contracts they cannot inspect
  * Security researchers spend hours reverse engineering
  * Attackers exploit this opacity



##   


## **What we built**

EVMDecompiler is an **AI-powered tool that turns unreadable bytecode into human-readable code — instantly**.

👉 [https://evmdecompiler.com/](https://evmdecompiler.com)

With one click, anyone can:

  * 🔍 Understand unverified smart contracts
  * 🧠 Recover logic using advanced AI models
  * ⚡ Analyze behavior in seconds instead of hours



We offer three models:

  * **Basic** — fast and accessible
  * **Pro** — higher accuracy and detail
  * **Max** — best performance for complex contracts



##   


## **Already used in the real world**

This is not a prototype — it is already widely used:

  * ✅ Integrated directly into Etherscan as a contract card
  * ✅ Serving **1000+ decompilations per day**
  * ✅ Used by auditors, researchers, and developers



This means users can directly inspect contracts while browsing, making Ethereum more understandable in practice.

##   


## **Why this matters**

Smart contracts control billions of dollars — yet most are effectively black boxes.

EVMDecompiler helps:

  * 🛡️ **Prevent scams** by exposing hidden or malicious logic
  * 🔍 **Speed up security analysis** and auditing
  * ⚡ **Enable faster incident response** during attacks



In short: **we make Ethereum actually transparent.**
